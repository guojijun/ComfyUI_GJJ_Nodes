# GJJ_TextImage

## 📋 概述

**功能**: 把文本渲染成图片，可用于标题卡、占位图、字幕图或提示词可视化。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_local_image_tools.py` | `GJJ_TextImage` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_TextImage` |
| **CATEGORY** | `GJJ/Image` |
| **FUNCTION** | `make` |

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
