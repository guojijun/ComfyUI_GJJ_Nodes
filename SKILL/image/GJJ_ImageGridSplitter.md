# GJJ_ImageGridSplitter

## 📋 概述

**功能**: 把图片按网格切成最多 9 块，可带少量重叠，适合局部处理后重组。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_image_grid_tools.py` | `GJJ_ImageGridSplitter` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_ImageGridSplitter` |
| **CATEGORY** | `GJJ/图像` |
| **FUNCTION** | `split` |

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
