# GJJ_RegionBox

## 📋 概述

**功能**: 创建一个可传递的矩形区域，并同步输出该区域遮罩。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_region_layer_tools.py` | `GJJ_RegionBox` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_RegionBox` |
| **CATEGORY** | `GJJ/Layer` |
| **FUNCTION** | `make_region` |

### 输入参数

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| `canvas_width` | `INT` | `1024` | ✓ |  |
| `canvas_height` | `INT` | `1024` | ✓ |  |
| `x` | `INT` | `0` | ✓ |  |
| `y` | `INT` | `0` | ✓ |  |
| `width` | `INT` | `512` | ✓ |  |
| `height` | `INT` | `512` | ✓ |  |
| `image` | `IMAGE` | `-` |  |  |

### 输出

| 输出名 | 类型 | 说明 |
|--------|------|------|
| 区域数据 | `REGION_TYPE` | |
| 区域遮罩 | `MASK` | |
| 区域JSON | `STRING` | |
