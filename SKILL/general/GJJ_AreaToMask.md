# GJJ_AreaToMask

## 📋 概述

**功能**: 按画布尺寸和矩形区域生成遮罩，也可直接接收 GJJ 区域数据。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_mask_toolkit.py` | `GJJ_AreaToMask` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_AreaToMask` |
| **CATEGORY** | `GJJ/Mask` |
| **FUNCTION** | `area_to_mask` |

### 输入参数

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| `canvas_width` | `INT` | `1024` | ✓ |  |
| `canvas_height` | `INT` | `1024` | ✓ |  |
| `x` | `INT` | `0` | ✓ |  |
| `y` | `INT` | `0` | ✓ |  |
| `width` | `INT` | `512` | ✓ |  |
| `height` | `INT` | `512` | ✓ |  |
| `blur` | `FLOAT` | `0.0` | ✓ |  |
| `region` | `REGION_TYPE` | `-` |  |  |

### 输出

| 输出名 | 类型 | 说明 |
|--------|------|------|
| 区域遮罩 | `MASK` | |
