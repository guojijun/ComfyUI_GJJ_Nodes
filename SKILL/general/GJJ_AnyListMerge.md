# GJJ_AnyListMerge

## 📋 概述

**功能**: 把多路任意输入合并成 ComfyUI 列表输出，适合批量参数、批量提示词和批量对象整理。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_any_list_tools.py` | `GJJ_AnyListMerge` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_AnyListMerge` |
| **CATEGORY** | `GJJ` |
| **FUNCTION** | `merge` |

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| `flatten` | `BOOLEAN` | `True` | ✓ |  |
| `skip_empty` | `BOOLEAN` | `True` | ✓ |  |
