# GJJ_MultiLoraChain

## 📋 概述

**功能**: 按配置顺序串联加载多组 LoRA，并根据是否接入 CLIP 决定只作用于模型还是同时作用于模型与文本编码器。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_multi_lora_chain.py` | `GJJ_MultiLoraChain` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_MultiLoraChain` |
| **CATEGORY** | `GJJ` |
| **FUNCTION** | `apply_loras` |

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
