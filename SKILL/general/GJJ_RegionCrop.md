# GJJ_RegionCrop

## 📋 概述

**功能**: 按 GJJ 区域数据从图片中裁切局部图像。

## 📁 文件映射

| 层级 | 文件 | 说明 |
|------|------|------|
| 🎨 前端 | `js/-` | 无前端文件 |
| 🔧 后端 | `nodes/gjj_region_layer_tools.py` | `GJJ_RegionCrop` 后端执行逻辑 |

## 🔧 后端节点

### 基础信息

| 属性 | 值 |
|------|-----|
| **类名** | `GJJ_RegionCrop` |
| **CATEGORY** | `GJJ/Layer` |
| **FUNCTION** | `crop` |

### 输入参数

| 参数名 | 类型 | 默认值 | 必填 | 说明 |
|--------|------|--------|------|------|
| `image` | `IMAGE` | `-` | ✓ |  |
| `region` | `REGION_TYPE` | `-` | ✓ |  |

### 输出

| 输出名 | 类型 | 说明 |
|--------|------|------|
| 裁切图像 | `IMAGE` | |
| 裁切遮罩 | `MASK` | |
