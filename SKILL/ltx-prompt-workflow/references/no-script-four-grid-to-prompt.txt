二、镜头连贯性与转场规则（核心优化）
2.1 连贯性推演逻辑

四个画面必须像水流一样推进，前一个画面的结束必须是后一个画面的起因或延续：

动作连贯（Match on Action）：上一格人物抬手，下一格顺势展现手部动作的结果。

视线连贯（Eyeline Match）：上一格人物看向某处，镜头顺着视线自然过渡（摇/移）到下一格的焦点。

物理空间连贯：通过镜头的持续推进（推）或后退（拉），在不切镜的情况下自然过渡景别（如：从远景一路匀速推进至特写）。

2.2 何时使用“切镜（Cut）”？

原则上优先使用连贯运镜，以下情况例外：

视角巨大转换：如从人物的主观视角（POV）突然跳跃到客观上帝视角。

时空/状态突变：室内到室外、白天到黑夜、平静突然转为极度爆发。

强烈对比需求：为了强调两个事物/两人的对立，直接使用“干净利落的硬切”。

三、四宫格叙事结构与台词控制

【起：场景渗入】(约3秒)：建立起始环境、光影与人物状态。镜头处于平稳或起始运动状态。（台词：0-1句，让画面先说话）

【承：顺势延展】(约3秒)：镜头顺着起步的动势、人物的走位或视线自然游走/推进，揭示更多细节或第二人物，情绪开始酝酿。（台词：1-2句，推进叙事）

【转：情绪聚焦点】(约3秒)：视线或镜头牢牢锁定核心冲突或情感爆发点（多为近景/特写），除非与上一画面反差极大才用硬切，否则用急推或持续聚焦。（台词：1-2句，核心对白）

【合：余韵收束】(约3秒)：动作完成，镜头随之停滞、缓慢拉开或定格。（台词：0-1句，或沉默）

四、音效设计规则（SFX Only）

严禁出现任何“背景音乐”、“BGM”、“钢琴曲”、“低沉弦乐”等描述。 听觉设计必须100%基于画面发生的物理现实：

环境音（Ambience）：风声、雨打玻璃声、远处的警笛、风吹树叶的沙沙声。

动作音/拟音（Foley）：脚步声的摩擦、衣物的窸窣声、咖啡勺碰触瓷杯的清脆声、沉重的呼吸声。

空间混响：空旷房间的回音、逼仄空间的沉闷感。

五、输出格式规则（严格执行）

一段式/两段式连贯自然语言 + 独立音效收尾（不要生硬地分成“第一段、第二段”来写，要当成一篇微小说/长镜头脚本来写）。
