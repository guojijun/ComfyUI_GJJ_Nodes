# GJJ_ImageToMask v1.1.0 功能增强总结

## 🎉 更新概览

本次更新为 `GJJ_ImageToMask` 节点添加了 **RGB单通道提取** 功能和 **全面的UI增强**，使节点从2种模式扩展到5种模式，并提供了更友好的用户界面。

---

## ✨ 核心改进

### 1️⃣ RGB单通道提取（新增3种模式）

#### 🔴 红色通道模式
```python
# 提取 R 通道 (索引 0)
mask = image[:, :, :, 0]
```
**适用场景**：
- 红外图像处理
- 热成像分析
- 红色标记检测
- 肤色分割辅助

#### 🟢 绿色通道模式
```python
# 提取 G 通道 (索引 1)
mask = image[:, :, :, 1]
```
**适用场景**：
- 绿幕抠图
- 植物/植被分析
- 自然场景处理
- 亮度信息提取（绿色通道通常包含最多亮度信息）

#### 🔵 蓝色通道模式
```python
# 提取 B 通道 (索引 2)
mask = image[:, :, :, 2]
```
**适用场景**：
- 天空/海洋分割
- 水体检测
- 冷色调分析
- 阴影区域提取

## 🎯 使用场景扩展

### 场景1: 颜色通道分析工作流
```
LoadImage
    ├→ GJJ_ImageToMask(💡 亮度) → PreviewImage (整体亮度)
    ├→ GJJ_ImageToMask(🔴 红色通道) → PreviewImage (R通道)
    ├→ GJJ_ImageToMask(🟢 绿色通道) → PreviewImage (G通道)
    └→ GJJ_ImageToMask(🔵 蓝色通道) → PreviewImage (B通道)
```
**用途**：分析图像各通道的分布和特征

### 场景2: 绿幕抠图优化
```
LoadImage(绿幕素材)
    → GJJ_ImageToMask(🟢 绿色通道) 
    → GJJ_MaskGrowBlur(扩张羽化)
    → Inpaint节点
```
**优势**：绿色通道通常包含最多的背景信息

### 场景3: 天空分割
```
LoadImage(风景照)
    → GJJ_ImageToMask(🔵 蓝色通道)
    → MaskToImage
    → 后续处理
```
**原理**：天空在蓝色通道中通常最亮

### 场景4: 红外图像预处理
```
LoadImage(红外图)
    → GJJ_ImageToMask(🔴 红色通道)
    → 生成热力图遮罩
```
**应用**：热成像、夜视图像处理

## 🔧 技术实现细节

### 智能Emoji解析
```python
# 用户看到："💡 亮度"
# 实际处理："亮度"
actual_method = method.split(" ", 1)[-1] if " " in method else method
```

### 条件分支扩展
```python
if actual_method == "Alpha通道":
    return (image[:, :, :, 3],)      # Alpha通道
elif actual_method == "红色通道":
    return (image[:, :, :, 0],)      # R通道
elif actual_method == "绿色通道":
    return (image[:, :, :, 1],)      # G通道
elif actual_method == "蓝色通道":
    return (image[:, :, :, 2],)      # B通道
else:
    return (_tensor_to_mask(image),) # 亮度模式（默认）
```

## 💡 最佳实践建议

### 1. 选择合适的模式
| 图像类型 | 推荐模式 | 原因 |
|---------|---------|------|
| 黑白线稿 | 💡 亮度 | 灰度转换最准确 |
| PNG透明图 | 🔳 Alpha通道 | 直接利用透明信息 |
| 绿幕素材 | 🟢 绿色通道 | 绿色背景最明显 |
| 蓝天照片 | 🔵 蓝色通道 | 天空在B通道最亮 |
| 红外图像 | 🔴 红色通道 | 热量在R通道最强 |

### 2. 组合使用
```
同时使用多个模式 → 对比分析 → 选择最佳结果
```

### 3. 配合其他节点
```
GJJ_ImageToMask 
    → GJJ_MaskGrowBlur (羽化边缘)
    → GJJ_MaskOutline (提取轮廓)
    → Inpaint节点 (局部重绘)
```
