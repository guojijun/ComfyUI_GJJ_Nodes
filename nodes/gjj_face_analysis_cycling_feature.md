# GJJ_FaceAnalysis 节点 - 源图轮询功能

## 功能改进

### 修改前
当源图和目标图都是批量输入时，采用**一一配对**：
- 4张目标图 + 2张源图 → 只处理前2张（min(4, 2) = 2）
- 目标图3、4被忽略

### 修改后
当源图和目标图都是批量输入时，采用**源图轮询**：
- 4张目标图 + 2张源图 → 处理所有4张
  - 目标1 → 源1
  - 目标2 → 源2
  - 目标3 → 源1（轮询）
  - 目标4 → 源2（轮询）

## 配对策略

### 1. 单源多目标
**保持不变**：同一张源图应用到所有目标图
```
源: [A]
目标: [1, 2, 3, 4]
结果: [A→1, A→2, A→3, A→4]
```

### 2. 单目标多源
**保持不变**：所有源图应用到同一张目标图
```
源: [A, B]
目标: [1]
结果: [A→1, B→1]
```
