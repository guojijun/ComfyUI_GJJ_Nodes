# GJJ_ImageToMask 快速参考

## 🎯 一句话介绍
🎭 将图像转换为遮罩，支持亮度、Alpha通道及RGB单通道五种模式。

## 📥 输入
- **🖼️ image** (IMAGE): 输入图像
- **🔄 method** (STRING): 
  - `💡 亮度`: RGB → 灰度遮罩
  - `🔳 Alpha通道`: RGBA → Alpha 遮罩
  - `🔴 红色通道`: 提取R通道
  - `🟢 绿色通道`: 提取G通道
  - `🔵 蓝色通道`: 提取B通道

## 📤 输出
- **遮罩** (MASK): [B, H, W] 张量

## 💡 使用技巧

### 场景1: 线稿转遮罩
```
选择"亮度"模式
白色区域 → 遮罩白色（选中）
黑色区域 → 遮罩黑色（未选中）
```

### 场景2: PNG透明图
```
选择"Alpha通道"模式
透明 → 遮罩黑色
不透明 → 遮罩白色
```

### 场景3: RGB通道分析
```
选择对应颜色通道模式
红色/绿色/蓝色通道 → 独立遮罩
用于颜色分离、通道混合
```

### 场景4: 配合重绘
```
LoadImage → GJJ_ImageToMask → Inpaint节点
```

## ⚡ 快捷操作
1. 搜索节点：`图片转遮罩` 或 `GJJ_ImageToMask`
2. 拖拽到工作流
3. 连接 🖼️ IMAGE 输入
4. 选择 🔄 转换方法（5种模式）
5. 获取 MASK 输出
6. 💡 提示：右键节点可快速切换模式

## 🔍 常见问题

**Q: 为什么我的 RGB 图像在 Alpha 模式下是全白？**  
A: RGB 没有 Alpha 通道，节点会返回全白遮罩。请使用"亮度"模式。

**Q: 遮罩值是 0-1 还是 0-255？**  
A: 0.0-1.0（浮点数），符合 ComfyUI MASK 标准。

**Q: 支持批量处理吗？**  
A: 支持！自动保持批次维度。
