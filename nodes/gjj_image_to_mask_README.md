# GJJ_ImageToMask 节点说明

## 功能概述

将图像（IMAGE）转换为遮罩（MASK），支持五种转换模式：

1. **💡 亮度模式**：根据图像的灰度值生成遮罩，适合黑白图、线稿等
2. **🔳 Alpha通道模式**：使用图像的透明通道作为遮罩，适合 PNG 透明图
3. **🔴 红色通道**：提取红色通道作为遮罩
4. **🟢 绿色通道**：提取绿色通道作为遮罩
5. **🔵 蓝色通道**：提取蓝色通道作为遮罩

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 🖼️ image | IMAGE | ✅ | 需要转换为遮罩的图像 |
| 🔄 method | STRING | ✅ | 转换方法：<br>- 💡 亮度：基于灰度<br>- 🔳 Alpha通道：基于透明度<br>- 🔴 红色通道：提取R通道<br>- 🟢 绿色通道：提取G通道<br>- 🔵 蓝色通道：提取B通道 |
| 📤 output_mode | STRING | ✅ | 输出方式：<br>- 🎭 仅遮罩：只输出 MASK<br>- 🖼️ 仅遮罩图：只输出 IMAGE<br>- ✨ 两者都输出：同时输出两种格式 |

## 输出

根据 **📤 输出方式** 的选择，动态返回不同格式：

| 输出模式 | 输出口1 | 输出口2 | 说明 |
|---------|---------|---------|------|
| 🎭 仅遮罩 | 🎭 遮罩 (MASK) | - | 只输出标准 MASK 格式 [B, H, W] |
| 🖼️ 仅遮罩图 | 🖼️ 遮罩图 (IMAGE) | - | 只输出可视化 IMAGE 格式 [B, H, W, 1] |
| ✨ 两者都输出 | 🎭 遮罩 (MASK) | 🖼️ 遮罩图 (IMAGE) | 同时输出两种格式 |

## 使用场景

### 1. 黑白线稿转遮罩
- 选择"亮度"模式
- 白色区域会变成遮罩的白色（选中区域）
- 黑色区域会变成遮罩的黑色（未选中区域）

### 2. PNG透明图转遮罩
- 选择"Alpha通道"模式
- 透明区域会变成遮罩的黑色
- 不透明区域会变成遮罩的白色

### 3. RGB通道提取
- 选择对应的颜色通道模式
- 适合分析特定颜色通道的信息
- 可用于颜色分离、通道混合等场景

### 4. 配合重绘节点使用
```
LoadImage → GJJ_ImageToMask → Inpaint/Paint nodes
```

## 技术实现

### 亮度模式算法
使用标准亮度公式将 RGB 转换为灰度：
```
grayscale = 0.299*R + 0.587*G + 0.114*B
```

### Alpha通道模式
直接提取 RGBA 图像的第4个通道（索引3）作为遮罩。

如果输入图像没有 Alpha 通道（RGB），则返回全白遮罩。

## 注意事项

1. **输出格式**：MASK 是 3D 张量 [B, H, W]，不是 4D 的 IMAGE
2. **数值范围**：遮罩值在 0.0-1.0 之间
3. **批量处理**：支持批量图像处理，保持批次维度不变
